Pràctiques C#
En aquest repositori hi han 3 pràctiques de C#:
1. Variables
2. Estructures
3. Strings, llistes i funcions

En aquestes 3 pràctiques es proposen diversos exercicis per a pràcticar la sintaxis de C#, com per exemple el calcul d'una equació de segon grau o fer el recompte de vocals dins d'una frase.
